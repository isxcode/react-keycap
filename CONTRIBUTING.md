### Installation Prerequisites

- Git 2.22.+
- Node 12.16.+
- Npm 6.13.+

### Build and Package

- for Npm

```shell script
git clone https://github.com/isxcode/react-keycap.git
cd react-keycap
npm install
npm run build
```

Note: If npm link in local, you should link your project react to react-keycap

```shell script
git clone https://github.com/isxcode/react-keycap.git
cd react-keycap
npm install
npm run add-react
npm run build
npm link
```
